# Assignment-01-A Simple

## Problem Statement

Create a responsive web page which shows an ecommerce admin dashboard with sidebar and statistics in cards using HTML, CSS and Bootstrap.

## What This Project Contains

- A fixed top navbar.
- A responsive sidebar menu.
- Four statistic cards for sales, orders, products and customers.
- Bootstrap grid classes for responsiveness.
- A small custom CSS file for layout and sidebar styling.

## Files

- `index.html` contains the dashboard structure.
- `css/style.css` contains the custom styling.

## How To Run

Open `index.html` in any web browser.

## How To Run In VS Code

1. Open VS Code.
2. Open the folder `Assignment-01-A-Simple`.
3. Open `index.html`.
4. Right-click and select **Open with Live Server**.

If Live Server is not installed, open `index.html` directly in a browser.

## Viva Questions

### 1. What is the aim of this assignment?

To create a responsive ecommerce admin dashboard using HTML, CSS and Bootstrap.

### 2. What is Bootstrap?

Bootstrap is a CSS framework used to create responsive and mobile-friendly web pages quickly.

### 3. What is a responsive web page?

A responsive web page adjusts its layout according to screen size, such as desktop, tablet and mobile.

### 4. Which Bootstrap classes are used for layout?

The project uses Bootstrap grid classes like `container-fluid`, `row`, `col-sm-6` and `col-xl-3`.

### 5. What is the use of cards in this project?

Cards are used to display ecommerce statistics such as total sales, orders, products and customers.

### 6. What is the use of sidebar?

The sidebar provides navigation links like Dashboard, Products, Orders and Customers.

### 7. How is the sidebar responsive?

On large screens, the sidebar is visible on the left side. On smaller screens, it opens as an offcanvas menu.

### 8. Which files contain your code?

`index.html` contains the dashboard structure and `css/style.css` contains custom styling.
